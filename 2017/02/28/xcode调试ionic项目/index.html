当使用node-sass时出现libsass bindings not found时

  • 先根据提示reinstall node-sass

    npm uninstall node-sass
npm install node-sass

  • 若是仍然有错误，重装所有node-moudles

    rm -rf node-moudles
npm install

  • 若是仍然有错误，重新build node-sass

    npm rebuild node-sass

  • 若是仍然有错误，则可能是gulp-sass的版本过低

    npm uninstall --save-dev gulp-sass
npm install --save-dev gulp-sass@2

gulp-sass是使用gulp构建项目时把sass编译成css的一个插件，当你使用node4.x及以上版本时，则会需求更高版本的gulp-sass
